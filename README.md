# Study Log App (study-log-ts)

日々の学習を記録する簡単な学習記録アプリです。

## 概要

このプロジェクトは、React と TypeScript を使用して構築されたアプリです。ユーザーは学習内容を記録し、日々の進捗を可視化することができます。UIコンポーネントライブラリとして Chakra UI v3 を採用し、モダンでレスポンシブなデザインを実現しています。バックエンド（データベース）には Supabase を使用しています。

## 主な機能

- **学習記録 (Study Records)**: 日々の学習内容、時間を記録・閲覧できます。
- **ダッシュボード (Home)**: ホーム画面です。
- **設定 (Setting)**: 設定ページです。機能なし。
